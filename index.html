Welcome to the style guide for Becky's Interior Design

Logo

The logo for Becky’s Interior Design is a simple black and white logo and shall stay that way. The text color can be changed to white if it is ever placed over a background where there is not enough contrast with the black text. See images below for examples.

The size of the logo shall never be larger than 200px width. It should always be placed in the left top corner in the header, and in the center of the footer. Margins around the logo should always be 10px.

10px

Typography

The Font used on Becky’s Interior Design website is always Mukta Light 300 for all text, and Poppins Medium 500 will be used for all headings. The text color should always be black, but it can change if more contrast is needed.

Layout

The website should always have 3 column grids, where the middle column covers 80% of the page and contains all content. The columns on the side should be the same size and work as margins to the webpage.

Interactive Features

There should be as little interactive features as possible to make the website appear minimalistic and modern. However it should contain image slider and buttons. The image sliders can only have a maximum of 5 images at a time and always have a button to toggle between the images.

Colors

There is one primary and one secondary color that should be used on the website. The colors are a monochrome yellow color, wich gives you a happy and comfortable feeling. These are the colors:

PRIMARY
Hex: #FFF4D3
SECONDARY
Hex: #FFE187

The primary color has the variable name primary-color and will be used as a background color. This is the color you will see covering the website.

The secondary color has the variable name secondary-color and will be used on all elements. Such as buttons, icons and links.

Form elements

All the forms should have the same styling. They should never be wider than 400px. Also they should have that minimalistic look so there should be no background color on the text elements. Only the buttons and interactive elements should have the secondary color as background color. See below for an example of a form.

Images and Media

Images will have no borders. Border radius will never be more than 10px. Hero-banners should always fill the page, and all other images should have the size of 350px x 350px. Other content areas

You should be able to style contents if you follow this style guide. However, other content areas like cards can have different styling. A card should always include an image, heading and text, button is optional. It should never be wider than 250px, and height should be set to fit-content. However, the height should be the same if there are several cards placed next to eachother. All call to action buttons should follow the rules for 'buttons' in interactive features.

The footer design should be the same on all pages. It should include the logo placed in the center, the address and social media links placed on the left side, and navigation on the right side.

Dos and Donts

Keep a read thread in the sizing of elements by using the sizes explained on each element. Do not make text smaller or larger than the default sizes.

Flex is key. A long text should not fill the width of the page. Keep it at 40% width with an element (preferably an image) next to it. Always use flex if there is room for more elements next to each other.

Branding

This is a professional website, however it is important that it is stylish and modern. We sell designs and our website should reflect the products we offer.

The target audience is men and women from the age 25+, interested in giving their home a new look or just need some inspiration.
